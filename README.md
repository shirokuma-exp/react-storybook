## 初期設定

git clone

npm init vite@latest tetris-vue -- --template vue-ts

npm install

<!-- vue-router の設定 -->
npm install vue-router@4

npm install -D eslint-plugin-vue @vue/eslint-config-typescript

npm install sass --save-dev

## 起動

npm run dev
